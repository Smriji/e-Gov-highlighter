# e-Gov 法令ハイライト

e-Gov 法令検索 "laws.e-gov.go.jp" の条文を読みやすくするために、キーワード、括弧、定義語を自動的に装飾（ハイライト）する Chrome 拡張機能です。

長く複雑な条文構成を視覚的に整理し、読解の負担を軽減することを目的としています。

## 主な機能

### キーワードのハイライト

* "及び"、"並びに"、"又は"、"若しくは" などの任意のキーワードに色付けや背景色を設定できます。

### 括弧のグレーアウト・網掛け

* 条文中の括弧（）とその中身を視覚的に目立たなくしたり、特定の色で網掛けしたりできます。

* 条文の見出し、"（１）" のような項目のナンバリングは装飾の対象外となるため、条文の構造を見失いません。

### 定義語のハイライト

* "以下「〇〇」という" や "「〇〇」とは、〜をいう" といった条文内の定義語を自動で抽出し、ハイライトや下線を引きます。

* "物"、"法"、"会社" といった定義語を除外するため、定義語の文字数の上限と下限を設定できます。

* "以下この条において「○○」という" などの適用範囲が制限されている定義は対象外です。

### 柔軟なカスタマイズ

* 拡張機能のオプション画面から、すべての装飾機能のオン/オフ、対象となる文字数、各種カラーコードなどを GUI で直感的に設定できます。

* 設定データは JSON 形式でエクスポート・インポートが可能です。

## インストール方法

現在、Chrome ウェブストアには公開されていません。以下の手順でローカルから読み込んで使用してください。

1. このリポジトリをクローン `git clone https://github.com/Smriji/e-Gov-highlighter.git` するか、必要なファイルを個別にダウンロードして任意のフォルダに展開します。

2. Google Chrome を開き、アドレスバーに **`chrome://extensions/`** と入力して拡張機能の管理画面を開きます。

3. 画面右上の **デベロッパー モード** をオンにします。

4. 画面左上の **パッケージ化されていない拡張機能を読み込む** をクリックします。

5. 手順 1 で展開したフォルダ（**manifest.json** が含まれているフォルダ）を選択します。

インストール後、e-Gov 法令検索のページを開くと自動的に装飾が適用されます。拡張機能のアイコンを右クリックし、**オプション** を選択すると設定画面が開きます。

## 設定のエクスポートとインポート

オプション画面の **詳細 / データ管理** タブから、現在の設定を JSON ファイルとして書き出すことができます。

他の環境に設定を移行したり、バックアップとして保存したりする際にご利用ください。インポートする際は、テキストエリアに JSON の内容を貼り付けて **変更を保存** をクリックしてください。

## 動作環境

* Google Chrome 最新版
* Microsoft Edge 最新版
* 対象サイト: e-Gov 法令検索 ("laws.e-gov.go.jp")
